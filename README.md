# Análisis de Ventas y Eficiencia Operativa - Walmart 🛒

## 🎯 Objetivo del Proyecto
Diseño de un sistema de reporteo ejecutivo para la Dirección Comercial con el fin de optimizar la asignación de presupuestos y la gestión de inventarios. El proyecto transforma datos semanales históricos en información estratégica mediante el cálculo de KPIs de rendimiento departamental.

## 🛠️ Herramientas y Tecnologías
* [cite_start]**Google Sheets:** Tablas dinámicas, fórmulas lógicas y validación de datos[cite: 5].
* [cite_start]**Procesos QA:** Ejecución de limpieza, enriquecimiento y aseguramiento de calidad sobre múltiples tablas transaccionales.
* [cite_start]**Dashboarding:** Construcción de visualizaciones interactivas y sparklines para monitoreo de KPIs.

## 📊 Metodología y KPIs Clave
* [cite_start]**Ventas por m²:** Evaluación de la eficiencia operativa ajustada al tamaño de la tienda[cite: 7].
* [cite_start]**Participación de Mercado:** Cálculo de la contribución relativa de cada departamento al total de ventas[cite: 7].
* [cite_start]**Comunicación:** Presentación de resultados bajo el método C→F→I (Contexto, Hallazgo, Impacto)[cite: 5].

## 📈 Logros e Impacto
[cite_start]Se desarrolló un dashboard dinámico que permite identificar departamentos con bajo potencial y áreas de alta eficiencia operativa, facilitando la toma de decisiones estratégicas basadas en datos reales[cite: 6].
